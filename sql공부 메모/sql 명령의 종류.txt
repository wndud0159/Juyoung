Structured Query Language 명령의 종류

-- 데이터 정의 언어(Data Definition Language) 테이블 구조를 설정한다.
-- CREATE, ALTER, DROP


-- 데이터 조작 언어(Data Manipulation Language) 테이블 내용을 검색, 입력, 수정, 삭제 한다.
-- SELECT, INSERT, UPDATE, DELETE


-- 데이터 제어 언어(Data Control Language) 데이터베이스를 관리한다.
-- GRANT, REVOKE


-- 트랜잭션 제어 언어(Transaction Control Language) 트랜잭션을 관리한다.
-- COMMIT, ROLLBACK
