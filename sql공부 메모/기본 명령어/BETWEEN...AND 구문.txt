BETWEEN...AND 구문
지정한 범위의 데이터를 검색한다.


 SELECT 검색필드명
 FROM   테이블명
 WHERE  필드명 [NOT] BETWEEN 시작값 AND 종료값;

사용 예
WHERE POINT BETWEEN 200 AND 400;
POINT 값이 200에서 400까지인 데이터 검색

WHERE CLASS_CD BETWEEN 'A' AND 'C'
CLASS_CD 값이 A에서 C까지의 데이터 검색


WHERE REG_DTTM BETWEEN SYSDATE AND SYSDATE+60
REG_DTTM 값이 현재일시에에서 현재일시 +60일까지의 데이터 검색

 고객 테이블에서 여성이고 누적포인트가 10,000에서 20,000이하인 고객을 검색한다.
SELECT *
FROM TB_CUSTOMER
WHERE MW_FLG = 'W'
AND TOTAL_POINT BETWEEN 10000 AND 20000;

 고객 테이블에서 생년원일이 19800101과 19891231사이인 고객을 검색한다
SELECT *
FROM TB_CUSTOMER
WHERE BIRTH_DAY BETWEEN '19800101' AND '19891231';

 고객 테이블에서 누적포인트가 10,000미만이거나 30,000이상인 고객을 검색한다
SELECT *
FROM TB_CUSTOMER
WHERE TOTAL_POINT NOT BETWEEN 10000 AND 30000;
