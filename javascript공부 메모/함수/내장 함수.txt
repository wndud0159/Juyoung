내장 함수

내장 함수는 자바스크립트에 기본적으로 내장되어 있는 함수들을 말한다

1. 인코딩, 디코딩 함수
http://icoxpublish.com/search?query=자바스크립트 와 같이 url주소에 쿼리 정보를
전소하여 데이터를 처리해야 되는 프로그램의 경우 한글과 같은 유니코드 문자가 
포함되어 있으면 오류가 발생할 수 있다
이런경우 인코딩 함수를 이용하면 문자를 부호화시키고 부호화된 문자를 다시
디코딩 함수를 이용하여 원래 문자로 되돌릴 수 있다
함수명
encodeURIComponent()    영문,숫자와() -_.~*!'을 제외한 문자를 인코딩한다
decodeUTIComponent()    encodeURIComponent()의 디코딩 함수


2. 숫자, 유/무한 값 판별 함수
함수명
isNaN()       숫자인지 아닌지를 판별하는 함수이다 숫자면 false반환 아니면 true반환
              NaN은 not a Number의 약자        
isFinite()    유한값인지 무한값인지 판변하는 함수이다 유한값이면 true반환, 무한값이면 false반환



3. 숫자, 문자 변환 함수
함수명
Number()      숫자로 변환해주는 함수
parseInt()    숫자와 문자가 포함되어 있을 경우 정수 부분만 숫자로 변환해 주는 함수
parseFloat()  숫자와 문자가 포함되어 있을 경우 소수 부분까지 숫자로 변환해 주는 함수
String()      문자로 바꾸어 주는 함수



4. 자바스크립트 코드 변경 함수
함수명
eval()        문자를 자바스크립트 코드로 변경해 주는 함수이다
