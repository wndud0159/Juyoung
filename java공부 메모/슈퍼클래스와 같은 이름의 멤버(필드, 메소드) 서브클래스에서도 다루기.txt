슈퍼클래스와 같은 이름의 멤버(필드, 메소드) 서브클래스에서도 다루기

class Car{
int x    ---- 슈퍼 클래스 멤버필드 변수 x
}

class RacingCar extends Car{
int x 	---- 서브클래스 멤버필드 변수x

public void show(){
	x = 10;   ----서브클래스 필드입니다.
	show.x = 10;  --- 슈퍼클래스 필드입니다.
	}
}

이처럼 서브클래스가 보유한멤버(메소드 및 필드)와 동일한 이름을 가진 슈퍼 클래스의 멤버에 접근하려면
멤버 이름앞에 'super.'을 붙입니다
