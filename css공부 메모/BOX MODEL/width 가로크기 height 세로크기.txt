width 가로크기 height 세로크기

요소의 가로크기, 세로크기를 말하며 기본적으로 여백과 테두리를 포함하지 않는다

속성값

width: 800px            요소의 가로 폭을 800px로 지정

height: 100%            요소의 높이 값을 100%로 지정

min-width: 1024px       요소의 가로 폭을 1024px 이상으로 지정

min-height: 200px       요소의 높이 값을 200px 이상으로 지정

max-width: 414px        요소의 너비 값을 414px 이하로 지정

max-height: 591px       요소의 높이 값을 591px 이하로 지정
